Teamstimmung (TS)

Die Teamstimmung (TS) beeinflusst direkt die Stärke des Mittelfelds und gehört damit zu den wichtigsten Faktoren für die Gesamtleistung einer Mannschaft. Der Ausgangswert zu Saisonbeginn liegt standardmäßig bei ruhig (TS 4,5). Mit einem Sportpsychologen kann dieser Ruhewert dauerhaft auf bis zu TS 5,0 erhöht werden.

Die TS wirkt sich auf die Spielaufbau-Leistung aller Spieler aus, die zum Mittelfeld beitragen. Je höher die Teamstimmung, desto stärker fällt das Mittelfeldrating aus.

Als grobe Orientierung gilt:

  • Jede TS-Stufe über „ruhig“ bringt etwa +7 % auf die Mittelfeldratings.
  • Jede TS-Stufe unter „ruhig“ verursacht etwas mehr als -7 % Abzug.

Die Teamstimmung verändert sich durch verschiedene Einflüsse:

  • Führungsqualitäten des Trainers
    Ein Trainer mit hoher Führungsqualität stabilisiert die Teamstimmung und sorgt dafür, dass sie langsamer sinkt.
  • Spieleinstellung
    Die gewählte Match-Taktik hat großen Einfluss:
    • Kraft sparen (PIC) erhöht die TS um ca. 33 %.
    • Spiel des Jahres (MOTS) senkt die TS um ca. 50 %.
  • Transfers
    Spielertransfers können die Teamstimmung negativ beeinflussen.
  • Sportpsychologen
    Sportpsychologen helfen dabei, die TS auf einem höheren Niveau zu halten.
Hinweis:
Die tatsächlich erreichbare TS vor Liga-/Pokalspielen
kann je nach Land, Updatezeiten und Transfers leicht abweichen.

Beispiel zur Berechnung der versteckten Teamstimmung (TS)

Im Aufstellungssimulator werden nur ganzzahlige TS-Werte zur Auswahl angezeigt. Intern arbeitet Hattrick jedoch mit genaueren Zwischenwerten. Diese können über die Änderung des Mittelfeldratings (MF) näherungsweise berechnet werden.

Beispiel

  • TS „gut“ = 6,5 → MF = 10,50
  • TS „ausgezeichnet“ = 7,5 → MF = 11,25

Der Unterschied von 1,0 TS-Punkt bewirkt also eine Änderung von 0,75 MF-Punkten.

Wird im Sandbox-Modus für die aktuelle Fähigkeitsstufe ein MF-Wert von 10,75 angezeigt, ergibt sich:

1. Differenz berechnen

10,75 − 10,50 = 0,25

2. TS-Anteil per Dreisatz berechnen

TS-Anteil = 0,25 / 0,75 = 0,333

3. Anteil zur Ausgangs-TS addieren

6,5 + 0,333 = 6,83

Ergebnis

TS ≈ 6,83

Diese Berechnung basiert auf linearer Interpolation zwischen zwei bekannten TS-/MF-Werten.



TI-Schraube (Trainingsintensität)

Eine kurzfristige Möglichkeit zur Erhöhung der Teamstimmung ist das Senken der Trainingsintensität (TI). Grundsätzlich gilt:

  • Niedrigere Trainingsintensität → höhere Teamstimmung
  • Höhere Trainingsintensität → niedrigere Teamstimmung

Diese Methode wird häufig für besonders wichtige Spiele genutzt, beispielsweise Relegationsspiele oder entscheidende Pokalpartien.

Wichtige Nachteile:

  • Verlust von Trainingseffektivität
  • Risiko eines Formeinbruchs
  • Negative Auswirkungen auf die langfristige Hintergrundform

Die TI-Schraube sollte daher nur in Ausnahmefällen eingesetzt werden.

Zeitpunkt der Anpassung

Damit die Maßnahme wirkt, muss die Trainingsintensität vor dem letzten täglichen Update vor dem Trainingsupdate gesenkt werden. Um Trainingsverluste möglichst gering zu halten, empfiehlt es sich, die Intensität direkt nach dem Spiel wieder anzuheben.
